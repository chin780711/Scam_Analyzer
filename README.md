# Scam Analyzer

Scam Analyzer 是一個以 **文字規則分析 + 網址風險判斷 + 可讀化解釋** 為核心的 AI 防詐騙 MVP 專案。  
使用者可以貼上可疑訊息、Email 內容或網址，系統會根據目前的規則與分析邏輯，提供風險等級、可疑重點、建議動作，以及網址分析結果。

> **注意：本專案目前仍為 MVP / 開發中版本，功能尚未完整，判斷結果僅供參考，後續仍會持續補強與優化。**

---

## 專案目標

這個專案的目標不是單純輸出一個分數，而是希望做到：

- 幫使用者快速辨識可疑訊息
- 分析網址是否存在假冒、相似網域、短網址、IP 網址等風險
- 以較容易理解的方式解釋「哪裡可疑」
- 提供對應風險等級的建議動作
- 作為後續整合 AI 語意分析與網頁內容分析的基礎版本

---

## 目前功能

### 1. 可疑文字內容分析
可分析內容中是否包含：

- 催促語氣
- 要求提供帳號、密碼、驗證碼
- 要求付款或金流資訊
- 假冒官方或客服
- 包裹 / 物流詐騙
- 求職詐騙
- 中獎誘導
- 附件誘導
- 電話號碼與施壓內容

### 2. 網址風險分析
目前支援分析：

- 非 HTTPS
- IP 位址型網址
- 短網址
- 過長網址
- 過多子網域
- Punycode 偽裝風險
- 高風險關鍵字
- 假冒品牌
- 相似網域偽裝（lookalike domain）

### 3. 使用者可讀說明
系統會整理出：

- 風險等級
- 分析摘要
- 你要注意的重點
- 詳細說明
- 建議做法 / 立即行動 / 不要做的事

### 4. 歷史紀錄與回饋
目前支援：

- 分析結果儲存到本地資料庫
- 最近分析紀錄顯示
- 使用者回饋（判斷正確 / 風險偏低 / 風險偏高）
